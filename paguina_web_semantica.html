Módulo: III - Diseño Responsivo

Descripción del Curso

El objetivo de este curso es comprender los fundamentos de HTML como lenguaje de marcado, identificando su estructura básica, principales etiquetas y su función en la construcción de páginas web, aplicando los conocimientos en ejercicios prácticos usando Visual Studio Code y codepen.io. En este módulo nos enfocamos en el diseño responsivo usando Flexbox y Grid.
